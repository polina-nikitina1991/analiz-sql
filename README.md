1. Проект по изучению активности в игре
Задача: проанализировать влияние покупки внутрииигровой валюты на поведение и активность играков.
Действие: с помощью SQL  провела анализ нулевых покупок, платящих и неплатящих игроков, просчитала среднюю стоимость и самые популярные предметы покупки, проаналиировала время между покупками
Результат: количество неплатящих игроков значительно превышает платящих 82%. Неплатящие игроки в среднем совершают больше покупок на 12%, но  платящие игроки покупают более дорогие эпические предметы. Средняя суммарная стоимость покупок у платящих игроков выше.
